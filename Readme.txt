# INSTALLATION INSTRUCTIONS #                                                 #
#############################                                                 #
#                                                                             #
# 1) Unzip the ZIP file to a directory on your hard drive.                    #
# 2) Create a directory on your server for SuperDesk to reside in. If your    #
#    server only allows CGI scripts to be executed in a certain directory     #
#    e.g. (/cgi-bin) then create this new directory in it.                    #
# 2) Upload the install.cgi and SuperDesk.cgi files to the new directory in   #
#    ASCII mode.                                                              #
# 3) Upload the Archive, Private and Public directories (and their contents)  #
#    to the new directory on the server in ASCII mode. If your server does    #
#    not allow HTML and CSS files to be accessed from the new directory (e.g. #
#    if you created it in the CGI-BIN) then upload the Public directory to    #
#    somewhere else.                                                          #
# 4) Upload the Private.tar file to the new directory in BINARY mode.         #
# 5) CHMOD the Private and Public directories to 777.                         #
# 6) CHMOD the SuperDesk.cgi and install.cgi files to 755.                    #
# 7) Load up the URL to the install.cgi script in your browser.               #
# 8) Follow the on-screen instructions.                                       #
# 9) After installation is complete, delete the install.cgi and               #
#    Private.tar files if they haven't already been deleted.                  #
#                                                                             #
# If you want to use any of the utility scripts, create a directory called    #
# Utilities in the SuperDesk directory and upload the one(s) you want to      #
# that directory.                                                             #
#                                                                             #
